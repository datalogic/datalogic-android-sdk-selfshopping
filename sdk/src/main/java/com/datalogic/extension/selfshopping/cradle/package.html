This package contains the {@link com.datalogic.extension.selfshopping.cradle.CradleManager} used to obtain an instance of the specific {@link com.datalogic.extension.selfshopping.cradle.Cradle} available in the device.

You can check the {@link com.datalogic.extension.selfshopping.cradle.CradleType} enum in order to obtain the correct type from the {@link com.datalogic.extension.selfshopping.cradle.Cradle} interface.

Currently, the single sub package {@link com.datalogic.extension.selfshopping.cradle.joyatouch} contains the interfaces and data types related to the Joya Touch Cradle.
